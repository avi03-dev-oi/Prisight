# Prisight

Prisight is a modern full-stack web application designed to provide an efficient and user-friendly digital management experience. The platform focuses on clean UI, smooth workflows, and scalable backend architecture.

## Features

- Secure Authentication System
- Interactive Dashboard
- Responsive Frontend Interface
- REST API Integration
- Database Management
- Modular and Scalable Architecture

## Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- Python
- FastAPI

### Database
- SQLite / SQLAlchemy

## Installation

### Clone the Repository

```bash
git clone git@github.com:avi03-dev-oi/Prisight.git
cd Prisight
```

### Backend Setup

```bash
cd backend
pip install -r req.txt
uvicorn main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
